Trigger the discovery skill. Through guided questioning, first clarify the problem framing, the first-version system requirements, the top-level API or interaction sketch, and a baseline flow or high-level design sketch, then derive what the system "must do" and "must not do", producing a grounded system shape and a structured discovery.md.
